# Face Detection Program

This is a simple Python program that uses your webcam to detect faces.

It draws a green box around each detected face. It does not identify people by name.

## Setup

Open a terminal in this folder:

```bash
cd /Users/vansh/codebase/code-with-vansh/face-detection-program
```

Create a virtual environment:

```bash
python3 -m venv .venv
```

Activate it:

```bash
source .venv/bin/activate
```

Install the dependency:

```bash
pip install -r requirements.txt
```

Run the program:

```bash
python face_detect.py
```

Press `q` to quit the webcam window.
